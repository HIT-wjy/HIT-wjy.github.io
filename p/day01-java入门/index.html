2.搭建Java开发环境

  • Java的产品叫JDK（Java development kit）Java开发者工具包，必须安装JDK才能使用Java
  • LTS：long-term support 长期支持版 JDK 8,11,17,21
  • 在官网下载 https://www.oracle.com
  • 安装路径不要有中文和空格

Java 和 Javac

  • java 是执行工具，Javac是编译工具
  • Java程序是高级语言，计算机底层是硬件不能识别这些语言，必须先通过javac编译工具进行编译，然后再通过java执行工具执行才可以驱动机器干活

总结

3.cmd的使用

  • CMD就是在windows中。利用命令行的方式操纵计算机。

打开CMD

  • 使用win+r

  • 输入cmd

  • 默认users之下

  • 使用win+e可以打开我的电脑

CMD的常见命令

  • 盘符名称加冒号。可以切换盘符
  • dir 。查看当前路径下的内容（包括隐藏文件夹）
  • cd+目录 。 进入单级目录 如果会面的目录名太长，可以使用tab进行关联
  • cd.. 。 回退到上一级目录
  • cd 目录1\目录2 。 进入多级目录
  • cd * 。回退到盘符*目录
  • cls 。 清屏
  • exit。 退出命令行

使用CMD打开QQ

  • 该如何方便的打开经常打开的软件呢？如何在任意的目录下面输入qq.exe都能打开qq

    • 只要把qq的路径记录在电脑的某个地方 ——>环境变量。
    • 这样就会现在当前路径找，找不到就去环境变量里去找

  • 如何操作呢？

    • 我的电脑—-属性—高级系统设置—-高级—-环境变量—–系统变量–PATH—-编辑—-新建—-qq.exe所在的完整路径加入—–一直确定。将qq的路径记录在PATH中，这样找不到的时候就会在环境变量中挨个找
    • 注意！需要重新打开一个cmd窗口，用原先的窗口是不行的

  • **环境变量！！！！**我们想在任意的目录都可以打开指定的软件，就可以把软件的路径配置到环境变量中。

4.开发第一个程序HelloWorld

  • 编写代码–>编译代码–>运行代码
  • 代码文件全英文，首字母大写，满足驼峰模式，源代码文件的后缀必须是 .java

编写代码

  • 先照着抄
  • 代码文件名必须与代码里所谓的类名称一致

编译代码

  • 使用javac命令进行编译，编译通过会生成hello.class文件
  • 使用java命令进行执行，注意不是 java hello.class 而是直接java hello 这是规定

总结

5.程序的原理

6.JDK的组成

  • JVM（Java Virtual Machine）：Java虚拟机，真正运行Java程序的地方

  • 核心类库：Java自己写好的程序，比如System.out.println

  • JRE(Java Runtime Environment)：Java的运行环境 上面两个加起来

  • javac 编译代码，java运行代码，把class文件送到JJVM中，运行的时候可能会调用核心类库

7.Java的跨平台特性

  • 跨平台：可以轻松移植到其他系统
  • Java跨平台：一次编译，处处可用 Java为各种平台都提供了对应的JVM虚拟机，程序再虚拟机里运行
  • 好处：开发效率高，成本低

8.PATH环境变量

  • Path环境变量可用于记住程序路径，以方便再命令行窗口的任意目录下直接启动该程序.
  • 用户环境变量只对当前用户生效，系统环境变量对所有用户生效
  • 加入环境变量，当从当前文件夹找不到时，会去环境变量中去找

  • 较新的JDK安装的时候，会自动配置javac，java程序的环境变量

  • 注意：老版本的JDK在安装时不会自动配置Path环境变量，此时，需要自己配置Path环境变量

  • 建议为JDK再配置JAVA_HOME环境变量

    • JAVA_HOME：用于告诉操作系统JDK安装在了哪个位置（将来其他技术要通过这个环境变量找JDK
    • JAVA的家不带Bin
    • 为了方便维护，可以讲Path中的路径改为**%JAVA_HOME%**。这样以后改JAVA_HOME，path中也会自动改

12.Java基础语法详解

注释

  • 单行，多行，文档注释
  • 文档注释是/** 一般用在类上，方法上，就是很专业的。里面的内容可以自动提取到程序说明书中去。
  • Java进行编译后生成class文件，其中会把注释全部去掉

  • 生成程序说明书 javadoc -d api（文件夹名字） CommentDemo.java

字面量

  • 告诉程序员：数据在程序中的书写格式

变量

变量的定义和使用

  • 变量用来记住程序要处理的数据
  • 数据类型 变量名称 = 数据; int age =18;

  • 使用变量记住数据再处理，程序的维护性更好更灵活

  • 变量的执行原理

    • 就是内存中的一块区域，用来存储数据

变量的特点

  • 变量里的数据可替换

使用变量的注意事项

  • 变量从定义开始到}结束有效。同一个范围内，不能定义两个名称一样的变量、
  • 变量定义时可以不赋值，但是使用时必须有值

关键字与标识符

关键字

  • Java语言自己用到的一些词，有特殊作用的，我们称之为关键字，如：public,class,int ,double
  • 我们不能用来作为：类名，变量名

标识符

  • 标识符就是名字（如类名，变量名…..都是标识符）
  • 基本要求：一般由数字，字母，下划线等组成，不要用中文
  • 强制要求：不能以数字开头，不能是关键字，不能有特殊字符（%。&。#）
  • 建议
    • 变量名：首字母小写的驼峰模式 studyNumber
    • 类名：首字母大写的驼峰模式 HelloWorld
